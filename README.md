# TestApp4

## Project Description
This is a basic Streamlit application for testing purposes. It serves as a starting point for building more complex applications.

## Prerequisites
- Python 3.8 or higher
- Streamlit

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/gituserc1140/TestApp4.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TestApp4
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application using the following command:
```bash
streamlit run app.py
```
